# Hệ thống Quản lý Kho

Hệ thống quản lý kho cho công ty viễn thông, giúp quản lý hàng tồn kho, thiết bị và vật tư.

## Tính năng

- Đăng nhập, xác thực người dùng
- Quản lý thiết bị (thêm, sửa, xóa, tìm kiếm)
- Quản lý nhập xuất kho
- Báo cáo và thống kê

## Công nghệ sử dụng

- React
- TypeScript
- Material-UI
- Axios

## Cài đặt

1. Clone repository:
```bash
git clone https://github.com/khanhtrung2610/Project.git
```

2. Cài đặt dependencies:
```bash
cd Project
npm install
```

3. Chạy ứng dụng:
```bash
npm start
```
